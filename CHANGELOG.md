# Changelog

## 0.0.3 (June 23, 2013)

* Added command help.
* When a user attempts to modify a term and is rate limited, Lita no longer also displays the term's current karma.

## 0.0.2 (June 22, 2013)

* Added term linking.
* Added `karma modified` command.
* Added rate limiting via `config.handlers.karma.cooldown`.

## 0.0.1 (June 15, 2003)

Initial release.
